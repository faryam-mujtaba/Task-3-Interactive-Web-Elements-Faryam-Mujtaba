# Task 3 - Interactive Web Elements

This is Task 3 of my Frontend Development Internship at DecodeLabs.

## Project Title

Interactive Web Elements

## Description

This project is an interactive webpage created using HTML, CSS, and JavaScript. It includes buttons, toggles, dynamic content updates, dark/light mode, click counter, show/hide content, text changing, and project filtering using JavaScript DOM manipulation.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- CSS Grid
- Flexbox
- GitHub Pages

## Features

- Dark and light mode toggle
- Dynamic greeting message
- Click counter
- Show and hide content
- Change text using JavaScript
- Project filter buttons
- Responsive layout
- Clean and readable design
